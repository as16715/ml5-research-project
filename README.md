# 🧠 ML5 Research Project — Creative ML Starter Kit

> *A micro-project library of beginner-friendly browser sketches built with [ml5.js](https://ml5js.org/), developed as part of my work as a Machine Learning Community Research Assistant.*

---

## 📌 About This Project

This repository is the result of a research proposal focused on making machine learning **accessible, visual, and fun** for beginners. The goal is to:

**Build a Creative ML Starter Kit — 10 Tiny Projects Anyone Can Build with ml5.js.**

Each project teaches **one core ml5.js concept** through a self-contained, browser-based sketch. No servers. No complex setup. Just open and explore.

The kit is structured so that every concept includes **at least two projects**:
- 🟢 **Easy** — for absolute beginners, minimal code, maximum clarity
- 🔴 **Advanced** — for those ready to go deeper, with more interactivity or creative complexity

By the end, this repo serves as a **living micro-project library** that anyone can clone, remix, and learn from.

---

## 🎯 Goals

- Lower the barrier to entry for creative machine learning in the browser
- Demonstrate real ml5.js concepts through hands-on, visual examples
- Provide clean, well-documented starter code anyone can build on
- Support the broader ML community through open, accessible research

## 🚀 How to Use

### Option 1 — Run Directly in the Browser
Each project will include a standalone HTML file. Simply download the folder and open `index.html` in your browser. No installation needed.
```bash
# Clone the full repo
git clone https://github.com/YOUR-USERNAME/ml5-research-project.git

# Navigate into any project
cd ml5-research-project/01-face-mesh/advanced

# Open index.html in your browser
open index.html
```

### Option 2 — Run in p5.js editor
Copy and paste the .js file into you p5.js editor and run the project.
